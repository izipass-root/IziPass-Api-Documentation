Authentication

 

IziPass API uses token-based authentication. All requests require authentication token to be set in headers: 

curl "https://izipass.ru/api/v1/v1.0?{request paramenters}"
  -X GET
  -H "Authorization: Bearer {access token}"
  -H "Accept-Language: en"

Request body

Refresh access token

Refreshes the expired access token, using the refresh_token parameter received in the Get token request

 

API endpoint

POST /api/v1/token

 

Request Headers
Content-Type "application/json"
Request body
grant_type "refresh_token"
client_type "ThirdParty"
refresh_token refresh_token parameter got from Get token request, string 
curl "https://izipass.ru/api/v1/token"
  -X POST
  -H "Content-Type: application/json"
                                            
{
     "client_type" : "ThirdParty",
     "grant_type" : "refresh_token",

token_type "bearer"

API methods

Get airports by query

Returns available airports matched by search query.

 

API endpoint

GET /api/v1/airports?query={airport query}&pageSize={page size}&page={page}

 

Request Headers
Accept-Language 2 character language code (ISO 639-1), supported are "ru" and "en", default is "en"
Request parameters
query Airport search string, min length is 3 characters.
Search is performed among three-letter airport IATA code and airport's name, country, city in English and Russian languages.
Returned all airports that contain search string in any of the abovementioned fields.
pageSize Number of elements to be returned on a single response page, 10 by default, integer
page Current page, 1 by default, integer 
curl "https://izipass.ru/api/v1/airports?query=hrk&pageSize=10&page=10"
  -X GET
  -H "Authorization Bearer {access_token}"
  -H "Accept-Language: en"

 

Response parameters
query Airport search string sent in the request
page Requested page number or default, integer
pageSize Requested number of airports to be returned on one page or default, integer
total Total number of airports found by the query parameter regardless the page and pageSize parameters, integer
airports A list of airports found - see below
Airport parameters
id Airport ID, string
name Localized airport name (if available localized, else - in English), string
city Localized name of a city airport relates to (if available localized, else - in English), string
country Localized country name (if available localized, else - in English), string
iataCode Three-character IATA code, string 
{
	"query" : "hrk",
	"page" : 1,
	"pageSize" : 10,
	"total" : 1,
	"airports" : [
		{	
			"id" : "d8e2b06d-b8c1-89b0-3004-4793fdc08cad",
			"name" : "Ekaterinburg , Koltsovo Airport, Russia (SVX)",
			"city" : "Ekaterinburg",
			"country" : "Russia", 
			"iataCode" : "SVX"
		}
	]
}

Get available services at the airport

Returns full information about airport services including information about availability and language translations.

 

API endpoint

GET /api/v1/airports/{airport_code}/availableservices?currencyCode={currencyCode}

 

Request parameters
airport_code Airport ID, GUID or Airport IATA code, string
currencyCode Excepted service prices currency, string. Supported is"RUB", default is "RUB" 
curl "https://izipass.ru/api/v1/airports/d8e2b06d-b8c1-89b0-3004-4793fdc08cad/availableservices?currencyCode=RUB"
  -X GET
  -H "Authorization: Bearer {access_token}"
or 
curl "https://izipass.ru/api/v1/airports/SVX/availableservices?currencyCode=RUB"
    -X GET
    -H "Authorization: Bearer {access_token}"
Response parameters
servicesWithTranslations A list of localized services model, see below
Localized services model parameters
language 2 character language code (ISO 639-1), supported are "ru" and "en"
services A list of service model, see below
Service model parameters
id Service ID, GUID
category Service category: BusinessLounge, VipService, MeetAndGreet or FastTrackstring
serviceDirection Arrival/Departure/Transit, string
customsType The service may be available for one or more customs types: international/domestic/schengen/nonSchengen, object with boolean fields
terminalName Name of the terminal where the service is granted (As added by service provider), string
name Localized name of the service (if available localized, else - in English), string
shortDescription Localized short description of the service (if available localized, else - in English), string
description Localized description of the service (if available localized, else - in English), string
meetingPoint Localized meeting point for the service - could be non-applicable (if available localized, else - in English), string
serviceCurrencyCode Service prices currency code, string
cancellationFees A list of service cancellation fees, see below
price Service prices - see below
plannedPrice Planned service prices - see below
isPlannedPriceEnabled Whether the service has planned price enabled or not, boolean
isNonRefundable Whether the service can be canceled or not, boolean. Some airports generate their QR-Codes which can't be canceled once generated
infantAge Infant age object - see below
childAge Child age object - see below
adultAge Adult age object - see below
facilities A list of facilities available. For Business lounges only, see below
preOrderPeriodInMinutes "Advance Booking Period" - That requires the booking to be finalized a certain number of minutes before the service delivery time, int
weekSchedule 7-item list that represents a service schedule for a week, see below
unavailabilityPeriods A list of unavailability periods, see below
supplierName Official supplier's name, string.
suplierINN Supplier taxpayer number, string.
supplierVAT Supplier VAT type, integer. 1 - without VAT; 2 - VAT 0%, 3 - VAT 10%, 4 - VAT 10/110, 5 - VAT 20%, 6 - VAT 20/120
Cancellation fees parametrs
hoursBeforeServiceDateTime Hours before planned service date time, integer
feeInPercents Cancellation fee percent, integer
Example: If an order is cancelled 6 h. before service time, you will be charged a 75% fee.
Price parameters
adult Price for adults, decimal
child Child price, decimal
infant Infant price, decimal
PlannedPrice parameters
plannedPriceChangeDateLocal The new price is applicable for the orders dated by the "plannedPriceChangeDateLocal" or later in local time of the Airport, datetime in ISO 8601 notation
plannedPriceChangeDateUtc The new price is applicable for the orders dated by the "plannedPriceChangeDateLocal" or later in UTC, datetime in ISO 8601 notation
adult Planned price for adults, decimal
child Planned child price, decimal
infant Planned infant price, decimal
Age parameters
bottom Lower age bound in years, integer, may be null
Inclusive. If lower age for aduld is 16, a person who is 16 years old will pay full adult price.
upper Upper age bound in years, integer, may be null
Exclusive. If upper age for infant is 2, infant, who is 2 years and 1 month old is considered a child and cannot be paid for using price for infant.
Facility parameters
name Name of the facility, string
isPaid Whether the facility is paid or not, boolean
Weekday parameters
dayNumber Index number of a day in the week, int from 0 to 6
name Name of the day, from Monday to Sunday, string
isActive Whether the service works at this day or not, boolean
begin Service opening time, time in ISO 8601 notation
end Service closing time, time in ISO 8601 notation
Unavailability Period parameters
begin Start of the unavailability period, datetime in ISO 8601 notation
end End of the unavailability period, datetime in ISO 8601 notation

Get service price

Returns a total service price for selected number of people for choosen service.

 

API endpoint

GET /api/v1/services/{service_id}/price?adultCount={adults}&childCount={children}&infantCount={infants}&currencyCode={currencyCode}

 

Request parameters
service_id Service ID, GUID
adultCount Number of adults in the order, integer
childCount Number of children in the order, integer
infantCount Number of infants in the count, integer
currencyCode Expected result currency, string. Supported is "RUB", default is "RUB" 
curl "https://izipass.ru/api/v1/services/d8e2b06d-b8c1-89b0-3004-4793fdc08cad/price?adultCount=2&childCount=1&infantCount=0&currencyCode=RUB"
  -X GET
  -H "Authorization Bearer {access_token}"
Response parameters
price Total price for all people in the order, decimal
currencyCode Total price currency code(ISO 4217), string 
{
    "price" : 32.55,
    "currencyCode": "RUB"
}
