Web Widget

Last updated: March 16th, 2020

API Key

IziPass uses API keys to allow access to the APIs. You can get API key through your IziPass Agent account - Widget page.

Widget configuration

Basic setup

Paste the following line into your page:

Basic setup example
<script id="izipass-web-widget" src="https://izipass.ru/widget/Init?apiKey=[Your API Key]" type="text/javascript"></script>

Extended setup

Or add additional parameters to customize widget appearance and behavior:

Extended setup example

<script id="izipass-web-widget" src="https://izipass.ru/widget/Init?apiKey=[Your API Key]&airportIata=HRK&height=800&lang=ru&hideHeader=true&BackgroundColor=%23302E47&ButtonColor=%23F9D332&InputColor=%23FFFFFF&LinkColor=%230089FF&HeadingColor=%23FFFFFF&DefaultTextColor=%23000000&ErrorColor=%23FF0000&TimepickerBackgroundColor=%23CCCCCC&ServiceDescriptionColor=%23767676&LabelColor=%2395989A&ImportantColor=%23F9D332" type="text/javascript"></script>

Widget Parameters

# Parameter name Value format Default value Description
1 apiKey GUID [Your api key] Your public API Key
2 airportIata Three-letter IATA code None If you want to open widget not on airport search page, but on the list of service categories available for particular airport, set this parameter to IATA code of desired airport
3 category "VipService", "BusinessLounge", "FastTrack" None If you want to open widget on specific service category of airport. Works only in pair with parameter airportIata
4 lang "en", "ua" or "ru" en Widget display language
5 height Integer value None Widget is able to dynamically adjust to container you put it into, but it is possible to limit max widget height using this parameter
6 hideHeader Boolean value false Widget could be shown either with or without header and footer. They are turned on and off simultaneously. Header contains language selection dropdown and footer contains IziPass logo
7 backgroundColor Hex color code #FFFFFF Color used for widget background
8 buttonColor Hex color code #FF6609 Color used for all buttons and to highlight airport that will be chosen in airport selection dropdown
9 inputColor Hex color code #FFFFFF Color used as background for inputs
10 linkColor Hex color code #0089FF Color used for links
11 headingColor Hex color code #FF6609 Color used for headings
12 defaultTextColor Hex color code #333333 Default color for text
13 errorColor Hex color code #FF0000 Color used to indicate errors
14 serviceDescriptionColor Hex color code #767676 Color used for service short description on service list page
15 labelColor Hex color code #333333 Color used for labels
16 importantColor Hex color code #000000 Color used for important information: price, order number and booking code
