Links

Last updated: March 16th, 2020

Why use links?

Sometimes our partners can't make API integration or put IziPass widget into their systems in case of different reasons.

IziPass links can solve this problem, you can redirect the customer to our site with a preselected airport, or airport services category.

Link configuration

Setup

Fill additional parameters in the following link and use on your site:

Link template
 https://izipass.ru/[lang]/?apiKey=[apiKey]&airportIata=[airportIata]&category=[category]
Link examples
1. https://izipass.ru/en/?apiKey=c0eef780-d326-4e77-b12c-2b6202ce46b8
2. https://izipass.ru/en/?apiKey=c0eef780-d326-4e77-b12c-2b6202ce46b8&airportIata=HRK
3. https://izipass.ru/en/?apiKey=c0eef780-d326-4e77-b12c-2b6202ce46b8&airportIata=HRK&category=VipService

Link Parameters

# Parameter name Value format Default value Description
1 apiKey GUID None Your public API Key. Required parameter.
2 airportIata Three-letter IATA code None If you want to open site not on airport search page, but on the list of service categories available for particular airport, set this parameter to IATA code of desired airport
3 category "VipService", "BusinessLounge", "FastTrack" None If you want to open site on specific service category of airport. Works only in pair with parameter airportIata
4 lang "en", "ua" or "ru" en Page display language
